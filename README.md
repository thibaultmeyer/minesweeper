## Build & install from sources
To compile and install this project, you must ensure that make, cmake, gcc or clang and GTK development files are being
correctly installed.

    #> mkdir cmake-build
    #> cd cmake-build
    #> cmake -DCMAKE_BUILD_TYPE=Release ..
    #> make
    #> make install
